# 四个指标

| 口语描述             | 代码字段                 | 输出标签                                           |
| -------------------- | ------------------------ | -------------------------------------------------- |
| 找没找到             | `recallMain`             | 召回率(Recall@10)                                  |
| 能不能给出结果       | 由 `recallMain > 0` 计算 | 有效覆盖率 ✅ 新增                                 |
| 排第几               | `firstHitScore`          | 首位命中分(MRR@10)                                 |
| 【废弃】整体排序质量 | `rankingQuality`         | 排序质量(nDCG@10)                                  |
| 首条准确率           | `top1Acc`                | 首条准确率(Top-1) 推荐的第一条是不是最应该推的那个 |
| 乱不乱推             | `falsePositive`          | 误触率(FP)                                         |
